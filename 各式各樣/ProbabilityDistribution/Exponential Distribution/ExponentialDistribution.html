指數分佈計算器

1. 基本參數計算 (期望值、變異數)

公式：E[X] = 1/λ, Var[X] = 1/λ²

2. 單一臨界值機率 P(X < k) 與 P(X > k)

累積分布函數公式(CDF)：P(X ≤ k) = 1 - e^(-λk)

3. 區間機率 P(m < X < n)

公式：P(m < X < n) = P(X < n) - P(X < m)

什麼是微分:微分就是在一個函數中x增加了一個無限小的值y會增加多少或減少多少f(x+(1/Infinity))-f(x)，但是這樣給出的答案一定是無限趨近於0， 所以為了讓他有意義而不是無限趨近於0，當給出x是多少時我們計算他的切線斜率，也就是(f(x+無限小)-f(x-無限小))/((x+無限小)-(x-無限小))， 等於(f(x+無限小)-f(x-無限小))/(無限小*2)， (f(x+(1/Infinity))-f(x-(1/Infinity)))/(2/Infinity)。 舉例來說y=2x這個式子將2x對x進行微分，不論x是多少我們都會得到2這個結果。

function n(x) {
    return 2*x; //differential: 2
}

function m(x) {
    return x**2; //differential: 2x
}

function o(x) {
    return 3*(x**2)+4*(x**3); //differential: 6x+12(x^2)
}

function differential(fun,x){
    let small = 1/(10**9);
    return (fun(x+small)-fun(x-small))/(small*2);
}

積分就是加總，舉例來說x是1時y是2、x是2時y是6、x是3時y是9，那麼x從1到3的積就是2+6+9=17，在微積分中的積分，舉例來說有一個函數是f(x)， 那麼x從0到9的積分就是f(0)+f(0+無限小)+f(0+無限小+無限小)+f(0+無限小+無限小+無限小)......f(9)，也就是直到0一直加無限小等於9就結束。 但是這樣得出的結果會是趨近無限大，這沒有意義，所以為了讓他有意義，我們必須再把剛剛得出的結果乘以無限小，這樣才會得出一個有意義的數。 積分也可以說是該函數在xy圖中與xy軸的面積。

指數分布機率密度函數Probability Density Function公式：PDF(X = k) = λ * e^(-λk)， 而他的積分所得到的新函數就是累積機率函數(累積分布函數 Cumulative Distribution Function): P(X ≤ k) = 1 - e^(-λk)。 而累積分布函數微分就可以得到瞬間機率函數(機率密度函數Probability Density Function)： PDF(X = k) = λ * e^(-λk)。 而瞬間機率函數乘以X，X從0到無限的加總，就是從0到無限的積分，就是期望值(所有項目和項目機率的加權平均數，所有機率總和必須為1): E(X = 0~無限) = 積分(X = 0~無限)[x* (λ * e^(-λx))]，設λ=1.5: 
function EXPDF(x) {
    let lambda = 1.5;
	let P = (lambda * Math.exp(-1*lambda*x));
	if(P<0.00000000000000000001){return "Error";}
    return x*P;
}

function integral(fun,start,end,small){
    if(end<start){return "error: end must bigger than start";}
    //let small = 1/(10**9);
	let sum = 0;
	for(let i=start;i<=end;i+=small){
	    let E = fun(i);
		if(E=="Error"){break;}
	    sum+=E;
		//console.log(i,sum,(i-start)/(end-start));
	}
    return sum*small;
}

而變異數就是(X-期望值)^2和項目機率的加權平均數，也就是 (X-所有項目和項目機率的加權平均數)^2和項目機率的加權平均數，也就是 (所有項目-所有項目和項目機率的加權平均數)^2和項目機率的加權平均數(注意所有機率總和必須為1)，也就是 (所有項目-所有項目的期望值)^2和項目機率的加權平均數(注意所有機率總和必須為1)，也就是 (X-X取期望值)^2和項目機率的加權平均數(注意所有機率總和必須為1)，也就是 (X-X取期望值)^2再繼續取期望值，也就是 [(X-X取期望值)^2]取期望值，也就是 期望值((X-期望值(X))^2)，也就是 E((X-E(X))^2)，也就是 積分(X = 0~無限)[((x - X取期望值)^2) * 機率函數(X)]，也就是 積分(X = 0~無限)[((x - X取期望值)^2) * 指數分布機率密度函數(X)]，也就是 積分(X = 0~無限)[((x - X取期望值)^2) * (λ * e^(-λx))]，也就是 積分(X = 0~無限)[((x - 積分(X = 0~無限)[x* (λ * e^(-λx))])^2) * (λ * e^(-λx))]: 
 
function exponential_distribution_probability_density_function(lambda,x){
    let P = (lambda * Math.exp(-1*lambda*x));
	if(P<0.00000000000000000001){return "Error";}
	return P;
}
